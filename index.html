Clipboard Kit Plugin Documentation

1. Introduction

What is Clipboard Kit?

Clipboard Kit is a powerful Unreal Engine plugin designed to simplify clipboard operations within your Unreal Engine projects. With Clipboard Kit, you can effortlessly copy, paste, and check text content in the clipboard, enhancing the user experience and improving data handling in your applications.

Features and Benefits

  • Copy Text to Clipboard: Easily copy text to the clipboard for sharing or saving within your Unreal Engine project.
  • Retrieve Clipboard Text: Get text from the clipboard, enabling paste functionality within your applications.
  • Check Clipboard Content: Determine if specific text is present in the clipboard, allowing for responsive actions.
  • Clipboard Status: Check if the clipboard is empty, providing valuable input for conditional decisions.

2. Installation

Requirements

Before installing Clipboard Kit, make sure you meet the following requirements:

  • Unreal Engine 5.3.0 or later.
  • Windows development platform.

Installation Steps

To install Clipboard Kit, follow these simple steps:

  1. Download the Clipboard Kit plugin from the Unreal Engine Marketplace.
  2. Open your Unreal Engine project.
  3. Navigate to the "Edit" menu and select "Plugins."
  4. Click "Installed" in the left sidebar.
  5. Click the "Enable" button next to Clipboard Kit.
  6. Restart Unreal Engine to complete the installation.
